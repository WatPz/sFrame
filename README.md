<p align="center", style="font-size: 26px">sFrame - 让 Script 不再复杂</p>

#### 文件树

- *sFrame*
  - *cfg*
    - Settings.cfg
  - *libs*
    - sF_Example.lua
    - sF_HUD.lua
  - sF_init.lua
  - sFrame.lua

#### 为什么要使用这个框架？

- 效率的提高

  - **sFrame** 采用 <u>面向对象</u> 的思维，而原本的 **API** 都是采用 <u>面向过程</u> 的思维
    很明显，前者在开发上的 <u>效率优势</u> 是后者无法比拟的

    这样的方式更利于 <u>程序员</u> *回忆思路和改进代码*

  - **sFrame** 通过 *libs* 的形式，封装了大量 **API** ，同时添加了许多新功能

    *简化* 了原本 *繁琐* 的逻辑，*提高*  了编写效率
